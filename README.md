# AfterKey

**AfterKey** is a decentralized document storage and inheritance management platform. It allows you to securely store important documents on the blockchain, track inactivity, and automatically share access with designated beneficiaries.

---

## Features

- **Blockchain Storage**: Securely stores your documents on-chain for tamper-proof records.  
- **Inactivity Detection**: Monitors user activity and triggers actions if inactivity is detected.  
- **Automatic Sharing**: Automatically shares designated documents with your assigned beneficiaries.
- **Emergency contact**:3–8 contacts can verify user status.Even if one confirms the user is active, the transfer is paused.
- **Grace Period Protection**:A user-defined waiting period before any asset transfer (e.g., 7, 30, or 60 days).Users receive alerts via email/app to confirm they are active.If there is no response, emergency contacts are notified to verify the situation.Reduces the risk of mistaken asset release.
